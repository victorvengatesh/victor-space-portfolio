# Victor Space Portfolio

A cinematic, animated portfolio for **M. Victor Vengatesh**, built with React, TypeScript, Vite, React Three Fiber, Three.js and Framer Motion.

## Features

- Full-screen procedural 3D universe
- Animated planets, orbital moon, satellite, stars, particles and shooting stars
- Mouse-reactive camera movement
- Scroll progress, reveal transitions and animated HUD interface
- Responsive mobile navigation
- Resume-driven About, Skills, Projects, Experience and Contact sections
- Downloadable resume in `public/Victor_Vengatesh_Resume.pdf`
- Reduced-motion accessibility support

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite.

## Production build

```bash
npm run build
npm run preview
```

## Personalize

Edit:

- Portfolio content: `src/App.tsx`
- 3D universe: `src/components/SpaceScene.tsx`
- Visual theme and animations: `src/styles.css`

## Deployment

Deploy the generated `dist` folder to Vercel, Netlify, Cloudflare Pages or GitHub Pages.
