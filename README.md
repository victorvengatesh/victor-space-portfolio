# M. Victor Vengatesh — Japanese Temple Portfolio

An immersive, scroll-driven AI/ML portfolio set inside a cinematic Japanese temple world.

## Experience

- Three architectural scenes: torii entrance, waterfall temple walkway, and lantern courtyard
- Scroll-linked camera movement and scene transitions with Motion
- Natural light-pink sakura petals and atmospheric depth
- Resume-based About, Skills, Projects, Experience, and Contact chapters
- Featured work: ClauseAI, AI Resume Analyzer, and VisionNet
- Responsive navigation, motion controls, and reduced-motion support
- Downloadable résumé and professional links

## Stack

Next.js, React, TypeScript, Tailwind CSS, and Motion.

## Run locally

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

The repository is ready for direct Vercel deployment.
